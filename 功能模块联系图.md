## 🔗 信号与槽联系

### 设备管理信号

```python
# DeviceManagerV2 发出的信号
device_added.connect(on_device_added)       # 设备已添加
device_removed.connect(on_device_removed)   # 设备已移除
data_updated.connect(update_ui)             # 数据已更新
status_changed.connect(update_status)       # 状态已变化
```

### 报警系统信号

```python
# AlarmManager 发出的信号
alarm_triggered.connect(show_banner)        # 报警已触发
alarm_acknowledged.connect(update_ui)       # 报警已确认
rules_updated.connect(refresh_list)         # 规则已更新
```

### 主题切换信号

```python
# ThemeManager 发出的信号
theme_changed.connect(apply_stylesheet)     # 主题已变化
```

## 📝 快速参考

### 添加新设备类型

1. 修改 `device_types.json`
2. 更新 `DeviceTypeManager`
3. 添加对应 UI 组件
4. 编写测试用例

### 添加新协议

1. 继承 `ModbusProtocol`
2. 实现协议方法
3. 注册到 `DeviceFactory`
4. 添加协议测试

### 添加新报警类型

1. 扩展 `AlarmType` 枚举
2. 添加检查逻辑到 `AlarmManager`
3. 更新 UI 显示
4. 编写测试
