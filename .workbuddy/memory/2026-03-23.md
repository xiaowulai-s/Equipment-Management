# 2026-03-23 日志

## 界面UI重新开发

根据UI设计方案文档，重新开发了工业设备管理系统的界面UI。

### 主要改动

1. **main.py** - 主程序重构
   - 采用现代化深色主题配色（基于UI设计方案）
   - 左侧设备列表带状态指示灯
   - 顶部设备信息栏
   - 实时趋势图（支持温度、压力、气体三线显示）
   - 仪表盘区域（4个仪表）
   - 数据卡片区域（4个关键数据卡片，带趋势指示）
   - Modbus寄存器表格
   - 设备控制面板

2. **gauge.py** - 仪表盘优化
   - 现代化圆弧仪表盘设计
   - 根据数值显示不同颜色（绿/黄/红）
   - 添加发光效果和刻度线

3. **data_card.py** - 数据卡片优化
   - 添加趋势指示器（↑↓→）
   - 渐变底部装饰条
   - 统一的配色系统

4. **trend_chart.py** - 趋势图优化
   - 深色主题样式
   - 支持多条曲线显示
   - 现代化图例设计

5. **modbus_table.py** - 寄存器表格优化
   - 深色主题样式
   - 交替行颜色
   - 状态颜色编码
