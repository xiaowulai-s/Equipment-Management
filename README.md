# Equipment Management System

工业设备管理系统上位机 - Version 1.0

## 项目简介

基于 Python + PyQt5 构建的现代化工业监控界面，实现设备列表管理、实时数据监控、仪表盘展示和 Modbus 寄存器通信功能。采用配置驱动、分层架构和插件化设计，支持多设备并发管理和实时数据采集。

## 技术栈

| 技术 | 说明 |
|------|------|
| Python 3.10+ | 编程语言 |
| PyQt5 | GUI 框架 |
| Matplotlib | 图表绘制 |
| NumPy | 数值计算 |
| pymodbus | Modbus 通信 |
| SQLite | 数据存储 |

## 功能特性

### 界面模块

| 模块 | 功能 |
|------|------|
| 设备列表 | 左侧设备列表，支持多设备切换，显示在线/离线状态 |
| 实时趋势图 | 基于 Matplotlib 的温度/压力实时曲线，每秒更新 |
| 仪表盘 | 4个圆形仪表盘（SQ10/AR2/B/C），支持多颜色状态显示 |
| 数据卡片 | 3个关键数据卡片：温度、压力、气体浓度 |
| Modbus表格 | 寄存器地址/功能码/值/状态 完整展示 |

### 核心功能

- **深色主题 UI** - 工业风格蓝青色调，符合 Windows 11 Fluent Design 风格
- **实时数据更新** - QTimer 驱动，每秒刷新所有数据
- **自定义控件** - CircularGauge、RealTimeTrendChart、DataCard、ModbusRegisterTable 等
- **模块化设计** - 清晰的分层架构，各模块职责明确
- **配置驱动** - 支持通过配置文件定义设备和寄存器映射
- **多设备管理** - 支持同时管理多个设备，并发通信
- **Modbus 通信** - 支持 Modbus TCP 协议，实现寄存器读写

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行程序

```bash
python run.py
```

或使用虚拟环境：

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

## 项目结构说明

### 主程序 (main.py)

- `IndustrialMonitorApp` - 主窗口类，实现整体界面布局
- `initUI()` - 初始化用户界面
- `initData()` - 初始化设备数据
- `initTimer()` - 初始化定时器
- `updateData()` - 更新实时数据

### 自定义控件

| 文件 | 控件类 | 说明 |
|------|--------|------|
| gauge.py | CircularGauge | 圆形仪表盘，基于 QPainter 绘制 |
| trend_chart.py | RealTimeTrendChart | 实时趋势图，基于 Matplotlib |
| data_card.py | DataCard | 数据展示卡片，显示关键参数 |
| modbus_table.py | ModbusRegisterTable | Modbus 寄存器表格，显示寄存器状态 |

## 开发说明

### 虚拟环境

推荐使用虚拟环境隔离依赖：

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

### 配置文件

系统支持通过 JSON/YAML 配置文件定义设备和寄存器映射：

```json
{
  "产品型号": "PumpStation-A",
  "变量列表": [
    {"变量名": "温度", "地址": 0x0001, "类型": "float", "读写权限": "R", "缩放因子": 0.1},
    {"变量名": "压力", "地址": 0x0002, "类型": "float", "读写权限": "R", "缩放因子": 0.1}
  ]
}
```
