# 一、全局强制规则（所有提示词必须带）

👉 **这是最重要的一段（必须加）**

```
使用 PySide6 编写 UI，必须严格遵守以下规则：

【布局规则】
- 严禁使用 setGeometry、move、resize 进行布局
- 必须使用 Qt Layout（QVBoxLayout / QHBoxLayout / QGridLayout）
- 所有控件必须加入 layout，禁止悬空控件
- 主窗口必须设置 centralWidget 并使用主布局

【结构规则】
- UI必须模块化，每个功能封装为独立 QWidget 类
- 使用布局嵌套（主布局 + 子布局）
- 必须使用 QWidget 作为容器划分区域

【显示规则】
- 界面必须自适应窗口缩放
- 禁止控件重叠
- 设置 stretch 或 sizePolicy 防止控件被压缩或遮挡

【代码规则】
- 输出完整可运行代码
- 结构清晰（main + UI类 + 子组件类）

```
