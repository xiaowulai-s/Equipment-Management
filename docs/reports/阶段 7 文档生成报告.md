## 使用指南

### 查看用户文档

1. **在线查看**
   ```bash
   # 直接打开 Markdown 文件
   start docs/用户文档.md
   ```

2. **转换为 PDF**
   ```bash
   # 使用 pandoc 转换
   pandoc docs/用户文档.md -o docs/用户文档.pdf
   ```

### 生成 API 文档

1. **安装依赖**
   ```bash
   pip install -r requirements-docs.txt
   ```

2. **生成文档**
   ```bash
   cd docs
   python generate_docs.py --build
   ```

3. **查看文档**
   ```bash
   start docs/_build/html/index.html
   ```

4. **实时更新**（可选）
   ```bash
   pip install sphinx-autobuild
   sphinx-autobuild docs docs/_build/html
   ```

### 自定义文档

1. **修改配置**
   - 编辑 `docs/conf.py`
   - 调整主题、扩展等

2. **添加模块**
   - 在 `docs/index.rst` 添加新模块
   - 创建对应的 `.rst` 文件

3. **重新生成**
   ```bash
   python docs/generate_docs.py --build
   ```
