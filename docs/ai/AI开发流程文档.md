```
功能要求：
- 实现ModbusTCP协议类ModbusTCPProtocol
- 支持功能码FC01（读线圈）、FC02（读离散输入）、FC03（读保持寄存器）、FC04（读输入寄存器）
- 支持功能码FC05（写单个线圈）、FC06（写单个寄存器）、FC15（写多个线圈）、FC16（写多个寄存器）
- 实现连接验证（检查IP地址和端口合法性）
- 实现连接状态监控

UI要求：
- 无UI要求

约束条件：
- 必须使用socket实现TCP通信
- 必须实现ModbusTCP ADU（应用数据单元）封装
- 必须实现事务ID管理
- 必须处理连接异常和超时

扩展属性：
- 支持连接池
- 支持自动重连机制
- 支持通信日志记录
- 支持数据包嗅探

输出信息：
- 输出modbus_protocol.py完整代码（ModbusTCP部分）
- 输出ModbusTCP协议文档
- 输出单元测试代码
```

```
功能要求：

扩展属性：
```
