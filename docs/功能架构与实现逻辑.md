## 三、核心功能实现逻辑详解

### 3.1 设备连接流程

```
用户点击"连接"按钮
    ↓
_main_window_v2._toggle_connection_by_id(device_id)
    ↓
判断设备状态：已连接→断开 / 未连接→连接
    ↓
core.device.device_manager_v2.DeviceManagerV2.connect_device(device_id)
    ↓
获取设备配置 → 创建设备实例
    ↓
core.device.device_factory.DeviceFactory.create_driver(config)
    ↓
根据协议类型创建驱动：
    - Modbus TCP → TCPDriver(host, port)
    - Modbus RTU → SerialDriver(port, baudrate, ...)
    ↓
core.device.device_factory.DeviceFactory.create_protocol(config)
    ↓
创建ModbusProtocol(mode, unit_id)
    ↓
启动连接 → 更新状态 → 发送device_connected信号
    ↓
UI接收信号 → 更新设备状态显示 → 启动数据轮询
```

### 3.2 数据采集流程

```
DeviceManagerV2._poll_all_devices() [定时触发]
    ↓
遍历所有设备 → 检查轮询间隔
    ↓
Device.read_registers() / protocol.read_registers()
    ↓
ModbusProtocol发送读取命令
    ↓
驱动层发送数据 (TCP/Serial)
    ↓
接收响应 → 解析数据 → 返回寄存器值
    ↓
DeviceManagerV2._on_device_data_updated()
    ↓
持久化数据 → 检查报警规则 → 发送data_updated信号
    ↓
UI接收信号 → 更新仪表盘/数据卡片/寄存器表格
```

### 3.3 报警检测流程

```
设备数据更新 → AlarmManager.check_value(device_id, parameter, value)
    ↓
遍历该设备的所有报警规则
    ↓
AlarmManager._rule_matches(rule, value)
    ↓
比较数值与阈值 (HH/H/L/LL)
    ↓
判断是否触发报警（考虑死区）
    ↓
触发报警 → AlarmManager._trigger_alarm()
    ↓
创建Alarm对象 → 添加到活动报警列表
    ↓
AlarmNotificationService.notify()
    ↓
检查冷却时间 → 发送通知(弹窗/声音)
    ↓
UI显示报警 → 用户确认 → 更新报警状态
```

### 3.4 数据持久化流程

```
数据采集 → _persist_data(device_id, data)
    ↓
添加到数据缓冲区
    ↓
缓冲区满或定时触发 → _flush_data_buffer()
    ↓
批量写入数据库 (SQLite)
    ↓
HistoricalDataRepository.insert_batch()
    ↓
执行SQL插入 → 提交事务
    ↓
CleanupScheduler定时清理过期数据
```

## 四、信号与槽机制

### 4.1 设备相关信号

| 信号 | 发射者 | 接收者 | 用途 |
|------|--------|--------|------|
| `device_added` | DeviceManagerV2 | MainWindowV2 | 设备添加后更新列表 |
| `device_removed` | DeviceManagerV2 | MainWindowV2 | 设备删除后更新列表 |
| `device_connected` | DeviceManagerV2 | MainWindowV2 | 设备连接后更新状态 |
| `device_disconnected` | DeviceManagerV2 | MainWindowV2 | 设备断开后更新状态 |
| `data_updated` | DeviceManagerV2 | MainWindowV2 | 数据更新后刷新UI |
| `device_error` | DeviceManagerV2 | MainWindowV2 | 设备错误时显示错误 |

### 4.2 报警相关信号

| 信号 | 发射者 | 接收者 | 用途 |
|------|--------|--------|------|
| `alarm_triggered` | AlarmManager | MainWindowV2 | 报警触发时通知UI |
| `alarm_acknowledged` | AlarmManager | MainWindowV2 | 报警确认后更新状态 |
