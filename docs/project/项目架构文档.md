# 工业设备管理系统 - 完整项目架构

## 📋 项目概述

**项目名称**: 工业设备管理系统 (Equipment Management System)\
**版本**: v1.6.0\
**架构**: 四层解耦架构 (4-Layer Decoupled Architecture)\
**技术栈**: Python + PySide6 + SQLAlchemy + Modbus\
**开发日期**: 2026-03-28

***

## 🎯 核心类关系图

### Device 类层次结构

```
Device (core/device/device_model.py)
├── 属性：id, name, protocol_type, status, ...
├── 方法：connect(), disconnect(), read_registers(), write_register()
└── 状态：DISCONNECTED, CONNECTING, CONNECTED, ERROR
         ↓ 使用
ModbusProtocol (core/protocols/modbus_protocol.py)
├── 方法：read_holding_registers(), write_register()
└── 依赖：TcpDriver / SerialDriver
         ↓ 使用
CommunicationDriver (core/communication/)
├── TcpDriver (Modbus TCP)
└── SerialDriver (Modbus RTU/ASCII)
```

### Repository 模式

```
BaseRepository (core/data/repository/base.py)
├── 属性：_session (SQLAlchemy Session)
├── 方法：add(), delete(), update(), query()
└── 泛型：[T]
         ↓ 继承
DeviceRepository (core/data/repository/device_repository.py)
├── 方法：get_by_id(), get_all(), save(), delete()
└── 操作：DeviceModel
         ↓ 同理
AlarmRepository
AlarmRuleRepository
HistoricalRepository
```

### Manager 类

```
DeviceManagerV2 (core/device/device_manager_v2.py)
├── 单例模式
├── 管理多个 Device 对象
├── 定时轮询机制
└── 信号：device_added, device_removed, data_updated

AlarmManager (core/utils/alarm_manager.py)
├── 单例模式
├── 管理报警规则
├── 检查报警条件
└── 信号：alarm_triggered, alarm_acknowledged
```

## 🎓 设计模式应用

### 1. 单例模式 (Singleton)

```python
# DatabaseManager
db = DatabaseManager()  # 始终返回同一实例

# DeviceManagerV2
manager = DeviceManagerV2.get_instance()

# AlarmManager
alarm_mgr = AlarmManager.get_instance()
```

### 2. 工厂模式 (Factory)

```python
# DeviceFactory
device = DeviceFactory.create_device(config)
# → 根据 protocol_type 返回 ModbusTCP/RTU/ASCII 设备
```

### 3. Repository 模式

```python
# 数据访问抽象
device_repo = DeviceRepository(session)
devices = device_repo.get_all()
device_repo.save(device)
```

### 4. 观察者模式 (Observer)

```python
# Signal/Slot 机制
device.data_updated.connect(update_ui)
alarm_manager.alarm_triggered.show_banner()
```

### 5. 策略模式 (Strategy)

```python
# 不同协议实现
class ModbusTCP(ModbusProtocol)
class ModbusRTU(ModbusProtocol)
class ModbusASCII(ModbusProtocol)
```

2. **Repository**: 在 `core/data/repository/` 创建数据访问类
