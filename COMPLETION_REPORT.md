## 🔧 技术改进

### CSS变量统一化
```css
/* 旧方案 */
--bg-primary, --bg-secondary, --bg-tertiary

/* 新方案（标准化） */
--bg-base        /* 最深层（页面背景） */
--bg-raised      /* 标准卡片背景 */
--bg-overlay     /* 浮动层背景 */
--bg-hover       /* 交互反馈背景 */
```

### 背景色应用规范

| 使用场景 | 变量 | 颜色 (深色) | 颜色 (浅色) |
|---------|------|-----------|-----------|
| 页面背景 | --bg-secondary | #0F1419 | #F9FAFB |
| 卡片/按钮 | --bg-raised | #161B22 | #F6F8FA |
| 悬停状态 | --bg-hover | #21262D | #F0F2F5 |
| 下拉菜单/弹窗 | --bg-overlay | #1C2128 | #F3F4F6 |

### CSS变量使用
```css
/* ✅ 推荐 */
background: var(--bg-raised);
color: var(--text-primary);
border: 1px solid var(--border-default);

/* ❌ 避免 */

color: #E6EDF3;
border: 1px solid #30363D;
```

### 响应式设计
```css
/* 支持移动端 */
@media (max-width: 768px) {
  .preview-grid {
    grid-template-columns: 1fr;
  }

    display: none;
  }

/* 使用transform优化 */

```
